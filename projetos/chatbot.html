Chatbot

  • Demonstração

Este projeto de Chatbot utiliza um banco de dados vetorial para permitir a ingestão de documentos e a realização de consultas com alta precisão. Ele é composto por dois principais componentes: o de ingestão de PDFs e o de consulta (query).

Os principais objetivos do projeto incluem:

  • Ingestão de PDFs: Permitir o envio de arquivos PDF para serem armazenados em um banco de dados vetorial (Pinecone), facilitando a consulta de informações contidas nos documentos.
  • Consulta Rápida: O chatbot é capaz de responder a perguntas com base nos PDFs ingeridos, utilizando o banco vetorial para fornecer respostas precisas e rápidas.
  • Gerenciamento Eficiente: Centraliza o armazenamento e a consulta de informações, permitindo que os usuários recuperem dados específicos de maneira ágil.

No componente de Ingestão, os usuários podem enviar arquivos PDF, que são processados e armazenados em um banco de dados vetorial (Pinecone). Isso facilita a indexação e a futura consulta de informações.

Principais funcionalidades:

  • Envio de PDFs: Interface intuitiva para carregar arquivos PDF diretamente na plataforma.
  • Armazenamento em Pinecone: Após o envio, o conteúdo dos PDFs é processado e armazenado no banco de dados vetorial, tornando a busca mais eficiente.
  • Processamento Rápido: O sistema processa os documentos rapidamente, permitindo a imediata disponibilidade para consulta.

O componente de Query permite que os usuários façam perguntas diretamente ao chatbot. Ele utiliza o banco de dados Pinecone para buscar respostas nos PDFs ingeridos.

Principais funcionalidades:

  • Busca Inteligente: O chatbot utiliza um algoritmo de busca vetorial para localizar rapidamente as informações mais relevantes nos documentos ingeridos.
  • Respostas Precisas: As respostas são baseadas no conteúdo dos PDFs, garantindo precisão e relevância na resposta.
  • Interface Simples: A interface de consulta é fácil de usar, permitindo que os usuários façam perguntas e obtenham respostas de forma rápida.
